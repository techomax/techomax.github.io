  {% if page.lang == 'en' %} Payment Terms {% else %} Zahlungsbedingungen {% endif %}

{% if page.lang == 'en' %} For first transactions, there are generally 3 payment options, as listed below.

  • Option 1: 100% prepayment to ship the goods directly to your company's warehouse, or
  • Option 2: 10% prepayment to ship the goods "on-hold" to an independent logistics warehouse of your choice
  • Option 3: Pickup from our warehouse EXW after paying in full (possible to inspect the goods beforehand)
For transactions with regular clients we usually require a partial prepayment of 30% and we can ship the goods directly to the customer. {% else %} Für erste Transaktionen gibt es im Allgemeinen 3 Zahlungsmöglichkeiten, wie unten aufgeführt.
  • Option 1: 100% Vorauszahlung, um die Ware direkt an das Lager Ihres Unternehmens zu liefern, oder
  • Option 2: 10% Vorauszahlung für den Versand der Ware "on hold" an ein unabhängiges Logistiklager Ihrer Wahl.
  • Option 3: Abholung von unserem Lager (EXW) nach vollständiger Bezahlung (es ist möglich, die Ware vorab zu prüfen.)
Bei Transaktionen mit Stammkunden verlangen wir in der Regel eine Abschlagszahlung von 30% und können die Ware direkt an den Kunden versenden. {% endif %}

  {% if page.lang == 'en' %} Credit Line {% else %} Kreditlinie {% endif %}

{% if page.lang == 'en' %} We do not currently have a credit line. The customer must either pay in advance or pay the full amount on the day of delivery. {% else %} Wir verfügen derzeit nicht über eine Kreditlinie. Der Kunde muss entweder im Voraus bezahlen oder den vollen Betrag am Tag der Lieferung bezahlen. {% endif %}

  {% if page.lang == 'en' %} Insurance {% else %} Versicherung {% endif %}

{% if page.lang == 'en' %} All our shipments are automatically insured, so don't worry. Note that the purchase of goods "EXW (Incoterms 2010)" means that the customer pays for both transport and insurance. {% else %} Alle unsere Sendungen sind automatisch versichert, also keine Sorge. Beachten Sie, dass der Kauf von Waren "EXW (Incoterms 2010)" bedeutet, dass der Kunde sowohl den Transport als auch die Versicherung bezahlt. {% endif %}

  {% if page.lang == 'en' %} Shipping Duration {% else %} Transportzeit {% endif %}

{% if page.lang == 'en' %} Depending on the agreement, we can arrange either a next-day delivery or a longer transport (and thus a better price). The standard delivery time is 2 days, but express shipping is almost always an option. {% else %} Je nach Vereinbarung können wir entweder eine Lieferung am nächsten Tag oder einen längeren Transport (und damit einen besseren Preis) organisieren. Die Standardlaufzeit beträgt 2 Tage, aber Expressversand ist fast immer eine Option. {% endif %}

 Drop Shipping

{% if page.lang == 'en' %} As long as the delivery country is within the EU, the customer is a European company and the goods are shipped from Austria, we can do drop shipping. {% else %} Solange das Lieferland innerhalb der EU liegt, der Kunde ein europäisches Unternehmen ist und die Ware aus Österreich verschickt wird, können wir Drop-Shipping machen. {% endif %}
